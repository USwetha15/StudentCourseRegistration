# Student Course Registration System

A full-stack web application built with Spring Boot, MySQL and HTML/CSS/JS.

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Data JPA, REST API
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## Features
- Student Registration & Login
- Browse Available Courses
- Course Enrollment
- Admin Dashboard to view all enrolled students
- Responsive dark themed UI

## Setup
1. Create MySQL database: `course_db`
2. Update `application.properties` with your DB credentials
3. Run Spring Boot app on port `8090`
4. Open `index.html` in your browser
